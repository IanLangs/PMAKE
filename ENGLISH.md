# PMAKE

[menu](./README.md)

A format for defining a compiler or interpreter for your projects.

You can specify commands to compile using multiple compilation modes:

-a -b -c

and so on, up to -z.

It works like this:

-a

Compilation command A or null

-b

Compilation command B or null

-c

Compilation command C or null

etc.

## Compile

PMAKE &lt;type&gt; &lt;pmake config file&gt;

## Show command

PMAKE &lt;type&gt; &lt;pmake config file&gt; -s
